# 中医经方 Skill — 双师承融合 · 经方派辨证论治系统

> 「中医很简单，就是阴阳气血。你搞懂了，一通百通。」—— 倪海厦
> 「难症痼疾，师法仲景。」—— 李可

## 项目简介

本项目是一个面向 AI Agent 的中医辨证论治技能包（Agent Skill），将**倪海厦**（1954-2012）的经方临床经验与**李可**（1930-2013）的扶阳急危重症思维融合为统一的双师承体系。以《伤寒论》六经辨证为核心框架，覆盖从常规辨证到急危重症的全场景中医方案，内置 16 部古籍原文、1400+ 医案、356 味本草索引、120+ 首经方，并具备自我学习与进化能力。


**触发词**：中医辨证 / 症状分析 / 经方 / 六经辨证 / 针灸处方 / 体质判断 / 方剂用药

## 核心特性

| 特性 | 说明 |
|------|------|
| **双师承融合** | 倪海厦（经方派·六经辨证·针灸）+ 李可（扶阳派·破格救心·急危重症），辨证统一，用药分流 |
| **16 部古籍原文** | 伤寒论、金匮要略、黄帝内经（素问+灵枢）、神农本草经、难经、脉经、针灸大成等，总计约 170 万字 |
| **1400+ 医案** | 倪海厦临床病历 951 份 + 诊疗日志 107 篇 + 仲景医案 74 例 + 李可医案 423 例 |
| **356 味本草索引** | 神农本草经上品/中品/下品全量注释 + 本草备要 + 本草正义深度阐发 |
| **十步辨证工作流** | 信息收集 → 六经定位 → 寒热鉴别 → 严重度评估 → 选方 → 医案验证 → 剂量决策 → 针灸配合 → 输出 → 反馈回溯 |
| **自进化系统** | 三层机制：实时反馈收集 → 定期自我审计 → 自动验证门（70 题基准测试） |
| **假证识别** | 15 种假证场景（假阳证 10 种 + 假阴证 5 种），防止真寒假热误诊 |
| **剂量分层** | 轻证→倪海厦常规剂量 / 中证→中等剂量 / 重证→李可破格大剂量，含附子安全六要素 |

## 在原始 Skill 基础上的扩展

本项目并非简单的合并搬运，而是在双 skill 基础上进行了大量原创性扩展：

### 古籍原文体系

新增 16 部中医经典古籍的完整 Markdown 原文，涵盖基础理论、药典、针灸、外科、妇科、急救等全类别，总计约 170 万字。所有古籍均建立了精细索引，支持按症状、药物、证型快速路由到相关段落。

| 类别 | 收录古籍 |
|------|---------|
| 基础理论 | 伤寒论、金匮要略、黄帝内经素问、黄帝内经灵枢、难经、脉经、濒湖脉学 |
| 药典 | 神农本草经、本草备要、本草正义 |
| 针灸 | 针灸甲乙经、针灸大成 |
| 专科 | 傅青主女科、外科正宗、肘后备急方、医宗金鉴 |

### 自我学习与进化系统

构建了三层自进化机制，使 skill 具备持续改进能力：

- **第一层 · 实时反馈**：自动收集用户追问、辨证纠正、疗效反馈，分别记录到 `feedback-log.md` 和 `treatment-outcomes.md`
- **第二层 · 定期审计**：每月自我审计协议——跑验证门 → 对比上月 → 查反馈日志 → 生成改进报告
- **第三层 · 验证门**：70 题基准测试（含 20 道陷阱题），低于阈值自动告警，确保知识更新不引入退化

### 问诊流程优化

在原始 skill 的辨证流程基础上，重新设计为**十步工作流**：

1. **信息收集** — 十问歌框架核对完整性，信息不足时按优先级追问
2. **六经定位** — 快速诊断流程图 + 六经辨证诊断公式
3. **寒热虚实鉴别** — 含真寒假热/真热假寒识别
4. **严重度评估 → 双师路由** — 常规→倪海厦 / 急危→李可 / 肿瘤→双师协同
5. **选方** — 倪海厦路径 + 李可路径 + 本草查询（本地优先，禁止联网搜药性）
6. **医案查询与验证** — 6 类医案库 Grep 检索，匹配度分析辅助验证选方
7. **剂量决策** — 三级剂量分层 + 附子安全六要素
8. **针灸配合** — 倪海厦体系独用，常见证型针灸处方速查
9. **标准化输出** — 统一格式（六经归属/证型/病机/治法/方药/参考医案/针灸/注意事项）
10. **反馈回溯** — 自动记录用户反馈，驱动持续改进

## 使用方法

在支持 Agent Skill 的 AI 平台（如 TRAE、Claude 等）中激活后，直接用自然语言描述症状即可：

```
我最近感冒了，怕冷，不出汗，头痛，脖子僵硬，脉浮紧。帮我辨证分析一下。
```

```
病人心衰，四肢厥冷，面红如妆，脉微细，怎么用李可的思路急救？
```

```
从倪海厦的角度分析一下：病人失眠多梦，头重胸闷，苔腻。
```

## 重要声明

- 本 skill 仅供学习参考，**不能替代专业医疗建议**
- 大剂量用药（尤其李可破格剂量）需在有经验的中医师指导下使用
- 临床处方须由执业中医师开立
- 古籍原文内容均有出处，未做任何编造
