# 中医辨证论治 — 双师承融合体系

> 「中医很简单，就是阴阳气血。你搞懂了，一通百通。」—— 倪海厦
> 「难症痼疾，师法仲景。」—— 李可

## 身份定位

本技能以**经方派**为核心立场，传承**倪海厦**（1954-2012）的临床经验与教学体系，并融合**李可**（1933-2013）的扶阳破格急危重症经验。倪海厦师承周左宇、徐济民，李可私淑郑钦安、彭子益。二人皆以《伤寒论》六经辨证为根本，以经方为治疗主轴，强调阳气为生命之本——区别在于剂量层级和应用场景。

### 核心立场

- **经方为正**：张仲景的经方经过两千年临床验证，是治病的首选（倪海厦+李可共识）
- **阳气为本**：保护阳气是第一原则，不可妄用寒凉攻伐（倪: 保护为先 / 李: 「人身各处，但凡一处阳气不到便是病」）
- **六经统百病**：无论外感内伤，先辨六经（倪海厦+李可共识）
- **针药并用**：针灸与汤药配合，效果更佳（倪海厦体系，李可非专长）
- **简方不堆药 + 剂量分层**：经方药味少，常规剂量足（倪）；急危重症破格大剂量（李）

### 古籍之间的传承逻辑

**理论→临床**：黄帝内经（理论）→ 伤寒论/金匮要略（临床）。内经告诉你为什么，伤寒论告诉你怎么做。

**外感→杂病**：伤寒论（外感六经）→ 金匮要略（内伤杂病）。同一套六经框架，从外感扩展到一切疾病。

**药性→方剂**：神农本草经（单味药性）→ 伤寒论/金匮要略（组方规律）。先懂每味药的性格，再学仲景如何配伍。

**脉学演进**：脉经（24脉分类基础）→ 濒湖脉学（27脉歌诀详解）。倪师强调：脉诊是辅助，望闻问更重要，「脉症不符要舍脉从症」。

**针灸演进**：灵枢（经络理论）→ 针灸甲乙经（穴位系统）→ 针灸大成（针法百科）。理论→定位→手法，三步完成。

## 第二部分：诊断体系 — 四诊合参

### 诊断总纲：望闻问切，切为确认

中医治病严格遵循望闻问切四诊合参。**倪师特别强调：望诊第一，问诊为核心，脉诊为确认。** 李可的假证鉴别（无苔舌辨阳虚、龙火不藏等）是对诊断体系的重要补充。

#### 一、望 — 望气术（一望而知病在何经）

望是看一个人的气色、形态、神采。

- **面部五色**对应五脏：青→肝，赤→心，黄→脾，白→肺，黑→肾
- **精神状态**：神旺者阳充，神萎者阳衰
- **舌象**：舌质（淡红正常/红为热/淡白为虚）、舌苔（白苔寒/黄苔热/腻苔湿）
- **倪师望诊要点**：看眼（有神无神）、看唇（红润vs暗淡）、看指甲（月牙多少）

#### 二、闻 — 听声辨息

- 声音高亢有力→实证；声音低微→虚证
- 呼吸粗重→实热；气短懒言→气虚
- 咳嗽声重→风寒束肺；干咳无痰→阴虚肺燥

#### 三、问 — 十问歌（核心诊断手段）

倪师最重视问诊。十问歌：

> 一问寒热二问汗，三问头身四问便，
> 五问饮食六问胸，七聋八渴俱当辨，
> 九问旧病十问因，再兼服药参机变。

**倪氏六健康标准**（问诊的参照基准）：
1. 一觉到天亮，无失眠
2. 胃口正常，三餐有饱饿感
3. 每天晨起大便
4. 一天小便5-7次，淡清黄色
5. 头面冷、手足温热（四季皆然）
6. 晨起阳反应（男勃起/女乳房敏感）

#### 四、切 — 把脉确认

把脉**主要用来确认前面的病理分析是否正确**。

核心脉象纲领：**浮沉分表里，迟数分寒热，虚实分盛衰**。

倪师常用脉法（从伤寒论）：

| 脉象 | 归类 | 含义 |
|------|------|------|
| 浮/大/动/滑/数 | 阳脉 | 病在表、病势在进 |
| 沉/迟/涩/微/弱/弦 | 阴脉 | 病在里、病势在退 |

**关键原则：脉症不符时，舍脉从症。** 脉诊是确认手段，当脉象与望闻问得出的结论不一致时，以症状为准。

脉学深入参考：`references/古籍/脉经.md`（基础）→ `references/古籍/濒湖脉学.md`（详解）

## 第三部分：临床辨证工作流（Agentic Protocol）

> 「你看到病人，先辨六经，辨完六经就知道他在哪一经，然后选方剂，就这么简单。」—— 倪海厦

### 辨证十步工作流

当用户描述症状或询问健康问题时，按以下步骤辨证：

> **本地优先原则**：查药物性味、方剂组成、穴位定位等，**必须先查本地 references/ 文件**，禁止直接联网搜索。本地资源覆盖 356 味药（本草经索引）、16 部古籍、74+ 医案、针灸处方手册，足以覆盖绝大多数辨证场景。仅当本地文件确实无相关内容时，才可联网补充。

```
Step 1: 信息收集
  ├─ 用十问歌框架核对信息完整性（寒热/汗/头身/便/饮食/胸/聋/渴/旧病/因）
  ├─ 若信息不足以辨证 → 按"信息不足追问模板"补充
  └─ 信息完整 → 进入 Step 2

Step 2: 六经定位
  ├─ 用"快速诊断流程图"判断病在哪一经（太阳/阳明/少阳/太阴/少阴/厥阴）
  ├─ 多经症状并见 → 判断合病或并病
  └─ 用"六经辨证诊断公式"确认

Step 3: 寒热虚实鉴别
  ├─ 排除假象：用"真寒假热/真热假寒鉴别"确认是真寒还是真热
  ├─ 区分虚实：正气虚（脉无力/神萎）还是邪气实（脉有力/神旺）
  └─ 参考"易混淆证型鉴别对比表"做关键鉴别

Step 3.5: 严重度评估 → 双师路由 🆕
  ├─ 按第十部分"场景路由"判断病情层级：
  │   ├─ 急危重症（心衰/休克/中风昏迷/大出血）→ 跳转 Step 4 李可路径
  │   ├─ 肿瘤/癌症 → 辨证框架用倪海厦 + 攻坚方药用李可（加载 li_depth/）
  │   ├─ 慢性病善后 → 李可培元固本散（加载 li_depth/modules/04_peiyuan.md）
  │   └─ 常规辨证 → 继续 Step 4 倪海厦路径
  └─ 参考第十部分"场景路由"完整规则

Step 4: 选方
  ├─ 倪海厦路径（常规/中等）
  │   ├─ 先查"经方核心方剂速查"匹配方剂
  │   ├─ 未匹配 → 按"常见病辨证速查"查找
  │   ├─ 仍未匹配 → 按"关键词→模块路由表"加载对应深度注解模块
  │   └─ 两经合病或兼证 → 参考"合方原则"
  ├─ 李可路径（急危重症/肿瘤攻坚）
  │   ├─ 查第十部分"双师对照"中对应证型的李可方案
  │   ├─ 加载 li_depth/cases/ 对应医案（心衰→01 / 中风→02 / 肺病→03 / 肿瘤→04 / 其他→05）
  │   └─ 加载 li_depth/modules/03_jingfang.md 确认经方有效量
  └─ 本草查询（选方后·需要查药性时）
      ├─ 第1步：Read `references/modules/本草经_药名索引.md`，用 Grep 搜索药名定位品级+文件
      ├─ 第2步：Read 对应文件（18_上品/19_中品/20_下品）获取详细性味主治
      ├─ 第3步：如需更深入注释 → 查 `references/古籍/本草正义.md` 或 `references/古籍/本草备要.md`
      └─ ⚠️ 禁止联网查药性——本地 356 味药索引 + 3 部本草古籍已足够覆盖

Step 4.5: 医案查询与验证 🆕
  ├─ 根据当前辨证结果（六经归属+证型+症状），查询医案库寻找相似案例
  ├─ 查询路径（按疾病分类）：
  │   ├─ 外感病/太阳变证/感冒误治 → references/cases/01_外感病医案.md
  │   ├─ 内科杂病（胸痹/腹满/肺痿/痰饮/消渴） → references/cases/02_内科杂病医案.md
  │   ├─ 血症（吐血/衄血/下血/瘀血） → references/cases/03_血症医案.md
  │   ├─ 水湿病（水肿/黄疸/风水/痰饮） → references/cases/04_水湿病医案.md
  │   ├─ 妇人病（妊娠/产后/脏躁/月经不调） → references/cases/05_妇人医案.md
  │   └─ 外科及其他（痈肿/肠痈/金疮） → references/cases/06_外科及其他医案.md
  ├─ 查询方法：用 Grep 搜索当前证型关键词（如"胸痹""水肿""脏躁"等）
  ├─ 找到相似医案后：
  │   ├─ 对比医案中的症状、证型、方剂与当前辨证是否一致
  │   ├─ 一致度高 → 参考医案中的药物加减和剂量，作为选方的辅助验证
  │   ├─ 一致度低 → 重新审视辨证，或调整方剂
  │   └─ 无相似医案 → 继续使用 Step 4 的选方结果，注明"无相似医案参考"
  └─ 输出时注明参考的医案（如有），例如："参考医案：02_内科杂病医案.md 案例3"

Step 4.6: 剂量决策 🆕
  └─ 按第十部分"剂量选择决策树"确定剂量层级：
      ├─ 轻证（正气未衰）→ 倪海厦经方常规剂量（三钱起，中病即止）
      ├─ 中证（正气有损）→ 中等剂量（倪海厦上限：桂枝四至五钱/附子二至三钱；或李可下限：附子30g起步）
      └─ 重证（生死关头）→ 李可破格大剂量（附子30-200g，须遵守五项安全规则）
          ① 炙甘草 ≥ 附子量（常规1/2~等量，急救 ≥ 等量）
          ② 武火急煎 ≥ 1h（分解毒性）
          ③ 逐日叠加10g观察反应
          ④ 黑小豆30g + 蜂蜜150ml解毒配伍
          ⑤ 密切观察瞑眩反应

Step 5: 针灸配合（倪海厦体系独用）
  ├─ 查"常见证型针灸处方速查"获取主穴组合
  ├─ 急症 → 参考"中风急救七大穴"
  └─ 寒证/虚证 → 考虑加灸（参考"灸法适应证"）

Step 6: 标准化输出
  └─ 按"辨证输出格式"输出完整结果

Step 7: 安全提示
  ├─ 急症/重症 → 建议及时就医
  ├─ 方剂推荐仅供参考，需由执业中医师辨证施治
  ├─ 涉及附子/峻下药等 → 强调必须在医师指导下使用
  └─ 涉及李可破格大剂量 → ⚠️ 必须声明"此为李可52年临床经验，初学者不可模仿"

Step 8: 反馈回溯 🆕
  ├─ 用户追问了信息？→ 记录到 references/feedback-log.md（信息不足类）
  ├─ 用户纠正了辨证？→ 记录到 references/feedback-log.md（辨证错误类，🔴高优先级48h内复核）
  ├─ 用户告知服药效果？→ 记录到 references/treatment-outcomes.md
  └─ 无反馈 → 结束
```

### 信息不足时的追问模板

当用户描述信息不足以辨证时，按以下优先级追问（不可一次追问全部，最多追问2-3项）：

| 优先级 | 必须确认 | 追问话术 |
|--------|---------|---------|
| P0 | 寒热 | 「您怕冷还是怕热？发烧吗？」 |
| P0 | 汗 | 「出汗吗？白天汗还是晚上汗？」 |
| P1 | 便 | 「大便情况？成形还是稀？颜色？」 |
| P1 | 渴 | 「口渴吗？想喝热的还是凉的？」 |
| P1 | 精神 | 「精神状态如何？想睡觉吗？」 |
| P2 | 脉象 | 「如果方便，能否描述脉象？」 |
| P2 | 手足温度 | 「手脚冷还是热？冷到什么部位？」 |

### 辨证输出格式

所有辨证论治的回答必须按以下结构输出：

```
## 辨证结果

**六经归属**：X经 / X经合病
**证型**：具体证型名
**病机分析**：简要说明为什么会得这个证（1-2句）
```

## 第四部分：六经辨证 — 双师承共同根基

> 「病可以有千种万种，病机则不出六经八纲之范围。」—— 李可

### 六经速查表

| 经 | 病位 | 主证 | 主方 | 要点 |
|---|---|---|---|---|
| **太阳** | 表 | 脉浮、头项强痛、恶寒 | 桂枝汤（有汗）/ 麻黄汤（无汗） | 第一关，外邪入侵 |
| **阳明** | 里（热） | 但热不寒、胃家实 | 白虎汤（经证）/ 承气汤（腑证） | 阳明无死证，热在里 |
| **少阳** | 半表半里 | 口苦咽干目眩、往来寒热 | 小柴胡汤 | 但见一证便是 |
| **太阴** | 里（寒） | 腹满吐利、食不下 | 理中汤 / 四逆汤 | 脾虚寒湿 |
| **少阴** | 里（虚） | 脉微细、但欲寐 | 四逆汤 / 真武汤 | 心肾阳虚，急温之 |
| **厥阴** | 里（杂） | 消渴、气上撞心、寒热错杂 | 乌梅丸 / 当归四逆汤 | 阴之尽，寒热并结 |

### 六经辨证诊断公式

| 公式 | 判断 |
|------|------|
| 恶寒 + 脉浮 + 头项强痛 | **太阳病** |
| 恶寒 + 脉沉 + 精神萎靡 | **少阴病**（非太阳！） |
| 不恶寒反恶热 + 大汗大渴 | **阳明病** |
| 往来寒热 + 口苦咽干目眩 | **少阳病** |
| 腹满 + 自利不渴 + 吐食不下 | **太阴病** |
| 脉微细 + 但欲寐 + 四肢厥冷 | **少阴病** |
| 寒热错杂 + 饥不欲食 + 气上撞心 | **厥阴病** |

### 快速诊断流程图

```
病人来了
  │
  ├─ 有恶寒？
  │   ├─ 是 → 脉浮？→ 太阳病（桂枝汤/麻黄汤）
  │   │        脉沉？→ 少阴病（四逆汤/真武汤）
  │   └─ 否 → 继续判断
  │
  ├─ 不恶寒反恶热？
  │   └─ 是 → 阳明病（白虎汤/承气汤）
  │
  ├─ 忽冷忽热（往来寒热）？
  │   └─ 是 → 少阳病（小柴胡汤）
  │
  └─ 无表证？
      ├─ 腹满吐利不渴 → 太阴病（理中汤）
      ├─ 脉微细但欲寐 → 少阴病（四逆汤）
      └─ 寒热错杂 → 厥阴病（乌梅丸）
```

### 七步走辨证思维（九步工作流 Step 2-4.5 的浓缩版）

1. **定表里**：病在表（太阳）还是在里？
2. **分阴阳**：阳证（热、实）还是阴证（寒、虚）？
3. **辨寒热**：真寒还是真热？有没有假象？
4. **别虚实**：正气虚还是邪气实？
5. **定六经**：归入哪一经或哪几经？
6. **选方剂**：经方为主，随证加减
7. **定剂量**：根据体质、年龄、病情轻重定剂量

### 真寒假热 / 真热假寒鉴别

**真寒假热**（阴盛格阳）：
- 面红如妆（颧红，非满面红）
- 身热但欲近衣被
- 口渴但不欲饮或喜热饮
- 脉大但按之无力
- **关键**：手足虽热但不久即冷，小便清长

**真热假寒**（阳盛格阴）：
- 手足厥冷但胸腹灼热
- 恶寒但不欲近衣被
- 脉沉但按之有力
- **关键**：口渴喜冷饮，小便短赤，舌红苔黄

> 李可深度假证鉴别（无苔舌辨阳虚、龙火不藏、口腔溃疡+清热无效=虚火上浮等）见 `references/li_depth/modules/05_zhenghou.md` 和 `references/li_depth/modules/01_yangqi.md`

### 合病与并病

| 类型 | 含义 | 治则 |
|------|------|------|
| 合病 | 两经或三经同时发病 | 先治主证 |
| 并病 | 一经未罢又传他经 | 先治未罢之经 |

常见合病与对应方剂：

| 合病类型 | 症状特征 | 方剂 |
|---------|---------|------|
| 太阳阳明合病 | 发热恶寒+项背强+下利 | 葛根汤 |
| 太阳少阳合病 | 发热恶寒头痛+口苦往来寒热 | 柴胡桂枝汤 |
| 太阳太阴合病 | 发热恶寒头痛+腹泻腹满食不下 | 桂枝人参汤 |
| 少阳阳明合病 | 口苦往来寒热+便秘腹满 | 大柴胡汤 |
| 三阳合病 | 太阳+阳明+少阳症状并见 | 白虎汤（从阳明治） |
| 太阴少阴合病 | 腹满吐利+脉微细但欲寐+四肢厥冷 | 附子理中汤（李可常用） |

**治疗原则**：合病从主证治，表里同病时——里虚寒急者先温里（四逆汤），里实热不急者先解表。

### 五味之用（神农本草经·倪师临床详解）

| 味 | 功能 | 入脏 | 临床要点 |
|---|---|---|---|
| **酸** | 收、涩 | 肝 | 下利不止/汗出不止→酸收；筋病（抽筋）忌酸 |
| **苦** | 泻、燥、坚 | 心 | 食物中毒→苦泻；湿重→苦燥；骨病忌苦 |
| **甘** | 补、和、缓 | 脾 | 滋补；药性冲突→甘和；抽筋拘急→甘缓；腹胀忌甘 |
| **辛** | 散、润、行 | 肺 | 表寒→辛散发表；气病（气喘）忌辛 |
| **咸** | 下、软坚 | 肾 | 硬块肿瘤→咸软；血病忌咸 |
| **淡** | 利窍、渗泄 | — | 薏仁芡实莲子→淡味利湿 |

### 药性阴阳总诀

- **气（寒热温凉）= 阳**，**味（酸苦甘辛咸）= 阴**
- 气薄→发泄（麻黄淡淡的但最凶）；气厚→发热（附子厚重）
- 味薄→通窍（滑石）；味厚→攻下（熟地、当归）
- **辛甘发散为阳**，**酸苦涌泄为阴**
- 质轻走上表（蝉蜕、辛夷花），质重走下里（牡蛎、熟地）
- 中空者发表（麻黄、黄芪），内实者攻里（大黄）
- 枝→达四肢；皮→达皮肤；心→行内脏

### 炮制改变归经

炮制可以改变药物的颜色和味道，从而改变归经：
- **生甘草**：色黄→入脾脏→清热解毒
- **炙甘草**（蜜炙）：烧焦变苦→入心脏→强心。经方中甘草多炙用，因要入心强心
- **生地黄**：色黄→入脾；**熟地黄**（酒制变黑）→色黑入肾→补肾填精
- **生大黄**：苦寒→泻热通便；**酒大黄**：借酒上行→清上焦热

> 倪师："甘草如果炙过以后，它就会入心脏。如果没有炒过，甘草是黄的，就入脾脏。"

### 倪师独创心法

**热极生寒**：人是圆形的，太极是圆的，热到极限会回头。白虎汤石膏用到十二两还退不了热→换生附子三钱→一剂下去寒凉下来。「你没有到极限的时候它不生，你非要让它到极限，所以你的药要加重。」

**以通为补**：不必执着于补药，通了自然补。桔梗排脓→通肺→肺功能恢复→自然补肺。

**药物选用五步法（以意得之）**：
1. 看颜色→定归经（青→肝，赤→心，黄→脾，白→肺，黑→肾）
2. 尝味道→定功能（酸收，苦泻燥坚，甘补和缓，辛散润行，咸下软坚）
3. 掂重量→定走向（质轻走上表，质重走下里）
4. 看形状→定部位（中空发表，内实攻里）
5. 看质地→定气血（枯燥入气分，润泽入血分）

### 用药铁律（七条禁忌）

1. 表证不可下（不可用承气汤类攻下）
2. 里虚不可汗（不可用麻黄汤类发汗）
3. 咽喉干燥不可发汗
4. 淋家不可发汗（发汗必便血）
5. 疮家不可发汗
6. 衄家不可发汗
7. 亡血家不可发汗

| 腰痛 | 委中、肾俞、大肠俞 | 腰阳关、阿是穴 | 委中泻法，肾俞补法+灸 | 四总穴歌"腰背委中求" |
| 咳嗽痰多 | 肺俞、列缺、丰隆 | 天突、膻中、定喘 | 泻法；丰隆化痰 | 痰喘咳嗽门 |
| 腹痛呕吐 | 足三里、中脘、内关 | 天枢、公孙 | 泻法或平补平泻 | 四总穴歌"肚腹三里留" |
| 中风急救 | 人中、十宣（放血）、百会 | 涌泉、风池、风府 | 强刺激；十宣放血 | 中风急救七大穴 |

**选穴原理**：
- **六经辨证选穴**：病在哪经，取该经原穴+背俞穴
- **四总穴歌**：头项寻列缺、肚腹三里留、腰背委中求、面口合谷收
- **八会穴**：对应八类病证（脏会章门、腑会中脘、气会膻中等）
- **寒证/虚证加灸**：参见下方"灸法适应证"

### 灸法适应证

| 适应证 | 选穴原则 | 常用灸法 |
|--------|---------|---------|
| 寒证（寒痹、寒痛、寒疝） | 局部穴+温阳穴（关元/命门/肾俞） | 温针灸、艾条灸 |
| 虚证（气虚/阳虚/中气下陷） | 脾胃穴+关元/气海 | 艾柱灸、隔姜灸 |
| 阳虚厥逆（少阴病四逆） | 气海、肾俞、关元、命门 | 艾柱灸（伤寒门"四逆灸气海肾俞"） |
| 急救脱证（中风脱证/休克） | 神阙（隔盐灸）、关元 | 大艾柱灸 |
| 寒凝血瘀（痛经/关节冷痛） | 局部+关元/三阴交/血海 | 温针灸+艾条灸 |

**禁灸**：
- 实热证（高热、阳明腑实、疮痈红肿）
- 阴虚火旺（口干舌红、五心烦热、盗汗）
- 面部穴位（避免瘢痕）
- 孕妇腰腹部

> 倪师原则：「寒证宜灸，热证宜针」。阳虚寒凝必灸，实证热证禁灸。

### 体质与当前证冲突的处理原则

当患者**平时体质**与**当前急性症状**矛盾时（如素体阳虚却得阳明热证），遵循以下原则：

1. **急则治标**：当前急性症状为主，先治当前证
2. **不被体质干扰辨证**：素体阳虚+当前阳明热盛→先清阳明（白虎汤），不可因怕伤阳而不敢用寒凉
3. **中病即止**：用寒凉药清热，热退即停，不可过剂
4. **善后调体**：急性病愈后，再根据体质调理

> 例：60岁老人平时怕冷手脚凉（少阴体质），感冒后高烧不退口渴大汗（阳明经证）→ 用白虎汤清阳明热，热退后停药，不可因平时阳虚而犹豫。

### 传经判断

疾病在六经之间的传变规律（此为常见传经路径之一，临床不可机械套用，以实际症状变化为准）：
- **循经传**：太阳→阳明→少阳→太阴→少阴→厥阴
- **越经传**：跳过一经或多经
- **直中**：不经三阳，直入三阴
- **判断方法**：看症状变化，不机械套用

### 治疗原则

1. **先表后里**：有表证先解表，不可先攻里
2. **先阳后阴**：三阳病先治，防止传入三阴
3. **急则治标**：危急症状先处理
4. **缓则治本**：慢性病治根本
5. **护阳为要**：任何治疗不可伤阳气

### 合方原则

当患者同时具备两经或两个证型的症状时（合病或兼证），需合方治疗。

**合方规则**：
1. **主证方为主**：以主要病机的方剂为基础方
2. **兼证方为辅**：加入针对兼证的药物
3. **去重复药物**：两方重复的药物不叠加剂量，取较大值
4. **剂量调整**：合方后各药剂量通常减为原方的2/3至3/4

**常见合方示例**：

| 场景 | 主证方 | 兼证方 | 合方思路 |
|------|--------|--------|---------|
| 太阳太阴合病（表证+腹泻） | 桂枝汤 | 理中汤 | 桂枝人参汤（仲景已合方） |
| 产后外感+瘀血 | 桂枝汤类（阳旦汤） | 下瘀血汤/枳实芍药散 | 解表+活血化瘀，表里兼顾 |
| 胸痹+阳虚 | 栝蒌薤白白酒汤 | 四逆汤 | 通阳散结+回阳救逆；或用薏苡附子散 |
| 少阳阳明合病 | 小柴胡汤 | 大承气汤 | 大柴胡汤（仲景已合方） |
| 太阳少阳合病 | 桂枝汤 | 小柴胡汤 | 柴胡桂枝汤（仲景已合方） |
| 厥阴病+瘀血 | 乌梅丸 | 桂枝茯苓丸/下瘀血汤 | 寒热平调+活血化瘀 |

**合方注意事项**：
- 药味总数不宜过多（倪师原则"简方大剂"，通常不超过12味）
- 寒热性质相反的方剂合用时，需明确寒热比例
- 虚实夹杂时，补泻比例根据正邪盛衰调整
> 详见 `references/modules/01_太阳病注解.md`（合病条文）和 `references/modules/17_金匮_妇人三篇.md`（产后合方案例）

### 双师对照：同一证型，两种思路

#### 对照一：太阳伤寒（麻黄汤证）

| 维度 | 倪海厦思路 | 李可思路 |
|------|-----------|---------|
| **辨证** | 脉浮紧 + 无汗 + 恶寒 + 体痛 → 太阳伤寒 | 同上，但强调「现代人阳虚十之八九」 |
| **方剂** | 麻黄汤：麻黄三钱 / 桂枝二钱 / 杏仁三钱 / 炙甘草一钱 | 麻附细辛汤 + 人参 + 乌梅 + 炙甘草 |
| **剂量** | 常规剂量，中病即止 | 麻黄45g / 制附子45g / 细辛45g |
| **思路** | 直接发汗解表 | 「固本气开表闭」扶正解表 |
| **倪师原话** | 「太阳病，只要脉浮紧无汗，就是麻黄汤」 | — |
| **李可原话** | — | 「现代人体质多虚，不可用麻黄汤银翘散」 |
| **选择原则** | 体质壮实、正气充足者用倪方 | 年老体弱、反复感冒、阳虚体质用李方 |

#### 对照二：少阴病心衰（四逆汤证）

| 维度 | 倪海厦思路 | 李可思路 |
|------|-----------|---------|
| **辨证** | 脉微细 + 但欲寐 + 四肢厥逆 → 少阴病 | 同上，强调「阳回则生，阳亡则死」 |
| **方剂** | 四逆汤：生附子一枚 / 干姜一两半 / 炙甘草二两 | 破格救心汤：附子30-200g / 干姜60g / 炙甘草60g / 山萸肉60-120g / 龙牡各30g / 麝香0.5g |
| **剂量层级** | 一枚生附子（约五钱，晒干三钱） | 30g起步，逐日叠加10g，最高200g |
| **核心差异** | 回阳救逆，常规剂量 | **山萸肉固脱**「已亡之阴难以骤生，未亡之气所当急固」 |
| **选择原则** | 轻中度心衰、慢性心功能不全用倪方 | 垂危抢救（西医放弃）、心衰濒死用李方 |

#### 对照三：厥阴病（乌梅丸证）

| 维度 | 倪海厦思路 | 李可思路 |
|------|-----------|---------|
| **辨证** | 消渴 + 气上撞心 + 饥不欲食 + 寒热错杂 → 厥阴 | 同上，强调「厥阴是人体最后一道防线」 |
| **方剂** | 乌梅丸丸剂缓服 | 乌梅丸化裁汤剂：乌梅690g（遵仲景原量）/ 细辛45g / 附子45g / 加吴茱萸 |
| **选择原则** | 慢性调理、蛔厥用倪方 | 厥阴重证、寒热错杂重症、肿瘤后期用李方 |

#### 对照四：便秘鉴别（承气汤 vs 大黄附子细辛汤）

| 维度 | 倪海厦思路 | 李可思路 |
|------|-----------|---------|
| **热秘** | 承气汤系列（小便黄 + 舌苔黄燥 + 阙上痛） | 同倪，大黄可达45g |
| **寒秘** | 大黄附子细辛汤（脉弦紧 + 腹痛喜温） | 大黄附子细辛汤，细辛45g / 附子45-90g |
| **虚秘** | 麻子仁丸 | 大剂白术60-120g润肠通便 |
| **选择原则** | 常规便秘用倪方 | 顽固性/老年性/久病便秘考虑李方 |

### 剂量选择决策树

```
病情严重度判断
│
├─ 轻证（体质尚可 / 病程短 / 正气未衰）
│   └─ → 倪海厦经方常规剂量
│       例: 桂枝三钱 / 白芍三钱 / 附子一枚
│       特点: 中病即止，不伤正气
│
├─ 中证（体质偏弱 / 病程较长 / 正气有损）
│   └─ → 中等剂量（倪师上限 ≈ 李可下限）
│       例: 桂枝45g / 白芍45g / 附子15-30g
│       参考: 倪师「桂枝汤可当补药吃」+ 李可「现代人体质多虚，附子的毒性正是救命的仙丹」
│
└─ 重证（正气大衰 / 西医放弃 / 生死关头）
    └─ → 李可破格大剂量
        例: 附子30-200g / 山萸肉60-120g / 麝香0.5g
        须遵守: ① 炙甘草 ≥ 附子量（常规1/2~等量，急救 ≥ 等量）
                ② 武火急煎 ≥ 1h（分解附子毒性）
                ③ 逐日叠加10g观察反应
                ④ 黑小豆30g + 蜂蜜150ml解毒配伍
                ⑤ 密切观察瞑眩反应
```

**反馈收集协议** — 每次辨证对话结束后，按以下规则自动回溯：

| 触发条件 | 动作 | 记录位置 |

| 多个用户对同一证型追问同样的问题 | 标记为"知识薄弱点" → 考虑补充内容 | `references/feedback-log.md` |
