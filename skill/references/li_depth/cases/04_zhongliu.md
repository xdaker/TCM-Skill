## 医案1：恶性淋巴瘤

**来源**：《李可老中医急危重症疑难病经验专辑》
**患者**：女、65岁
**诊断**：左颈部弥漫型恶性淋巴瘤混合细胞型（病理号3054）
**症状**：颈左侧肿物40天，初起如黄豆大，未及1个月猛长如初生婴儿头大，向下蔓延至左锁骨上窝，凹凸如岩，坚硬不移；颈右侧及颊车穴下方肿块6个，大如杏核，连成一串；双腋下、双腹股沟淋巴结皆肿大如枣。头痛如破，气喘痰壅，胸部憋胀，面色灰滞，神识昏糊，二便闭结，溲若浓茶，口臭熏人。
**辨证**：痰毒弥漫三焦，毒入血分，阻塞气机，蒙蔽神明重症
**处方**：漂海藻、生甘草、煅礞石、木鳖子、生半夏、鲜生姜、莱菔子(生炒各半)、黄药子、鳖甲、生牡蛎、浮海石、海蛤壳、元参、蚤休各30克，大黄、大贝、桃杏仁各15克，山茨菇、山豆根、红花各10克，"全虫12只、蜈蚣4条、明雄黄1.2克"(研末冲服)
**剂量**：以蛇舌草、夏枯草各120克煎汤代水煎药，煎取浓汁600毫升，日分3次服，7剂
**疗效**：服首次药后约1刻钟出现瞑眩反应，吐出胶粘痰涎半痰盂，胸膈顿觉宽敞。服2~7剂时每日畅泻污泥状夹有脓血、胶粘痰涎之大便1~2次，尿已转清，头已不痛，神识清朗，食纳大增，全身肿块变软。经治2个月，服药34剂，临床缓解。至1981年春随访，体健逾于往年，已临床缓解3年半。
**李可按语**：本方脱胎于兰州已故名医董静庵先生之验方"海藻甘草汤"，原方主治瘰疬。我师董老意，加量3倍，虫类药研粉吞服，以加强药效。另加鳖甲、消瘰丸、夏枯草、生半夏、鲜生姜，大大加强了养阴化痰、攻坚散结之力。《内经》有"药不瞑眩，厥疾弗瘳"之记载，一旦出现瞑眩现象，必有非常之效。

## 医案2：甲状腺癌颈转移

**来源**：《李可老中医急危重症疑难病经验专辑》
**患者**：女、60岁
**诊断**：甲状腺癌颈转移（省肿瘤医院超声探查及同位素扫描确诊）
**症状**：颈部肿块29年，甲状软骨上方肿块杏子大，下方肿块约乒乓球大，均质硬，右颈部鹅蛋大肿块，凹凸不平。从8岁起抽旱烟，支气管炎30年。近3年暴喘迫促，两臂上举则气闭晕厥。痰声如拽锯，稠粘难出。目赤，胸胃烧灼难耐。日食冰棍1桶，水果罐头无数。放疗后耳聋不闻雷声。近2个月有血性涕，剧烈右偏头痛。
**辨证**：高年肾阴大亏，阴不抱阳，龙雷之火上燔，痰气交阻，日久化火化毒
**处方**：
- 方1（引火汤）：九地90克，盐巴戟肉、二冬各30克，云苓15克，五味子6克，上油桂2克（去粗皮研粉小米蒸烂为丸先吞），3剂
- 方2（攻癌夺命汤加减）：漂海藻、昆布、生半夏、鲜生姜、元参、花粉、海蛤壳、牡蛎、黄药子、木鳖子、蛇舌草、夏枯草、生苡仁、蚤休各30克，大贝、麦冬、桃杏仁各15克，白参（另炖）、五味子、山茨菇、山豆根各10克，竹沥2匙，"全虫12只，蜈蚣4条，上沉香1.5克，明雄黄1.2克"（研粉吞服）
**剂量**：头3个月每旬服7剂，至9月底共服70剂；至1979年6月共服百剂
**疗效**：全身脂肪瘤消失，右颈转移灶缩小2/3，甲状软骨上下之肿物亦明显缩小。血性涕消失，痰声漉漉偶见。动则暴喘之状可减三四。喘息已很轻微，可到邻家串门。至1980年3月，所有肿物全部消失。计经治18个月，服药300剂，其中引火汤约占1/4。现仍健在，已80高龄。
**李可按语**：凡见上热无制，即服引火汤3剂。治癌是持久战，正胜邪却，暂时的缓解，瘤体的消失，不等于癌毒的彻底消灭。一旦人体正气有亏，癌毒又成燎原之势。"炉烟虽熄，灰中有火"，故除恶务尽，不使死灰复燃。

## 医案3：胃小弯癌

**来源**：《李可老中医急危重症疑难病经验专辑》
**患者**：男、60岁
**诊断**：胃小弯癌（4×4cm），西安医学院二院病检确诊
**症状**：食入即吐，痰涎如涌。便燥，三五日一行，下结如羊粪球，落地有声。面色灰滞，消瘦，病未及3个月体重下降15公斤。然神识清朗，食欲颇佳，声若洪钟。一生嗜食肥甘，嗜酒如命。
**辨证**：湿热酿痰，阻塞气机，日久化毒，积为有形癥积。所幸正气未衰，可以用攻。
**处方**：赭石末50克，漂海藻、生甘草、元参、牡蛎、醋鳖甲、木鳖子、黄药子、生半夏、鲜生姜、蛇舌草、夏枯草、莱菔子各30克（生炒各半），旋覆花（包）、醋柴胡、山茨菇各15克，红参（另炖）、灵脂各10克，"全虫12只、蜈蚣4条，紫硇砂3克，明雄黄0.3克"（研末冲服）
**剂量**：煎取浓汁400毫升，对入蜂蜜100克、姜汁10毫升煎3沸，日分2次服，30剂。另隔日冲服儿茶2克。
**疗效**：服至5剂后，大便通畅，进食不吐，已与平日无异。服完汤剂，调养月余，在地区医院镜检，瘤体消失，食纳如常，体重恢复，已返陕照常参加农事劳作。
**李可按语**：胸怀豁达，便易措手。胃癌之呕吐，多兼见大便燥结，此为痰毒结于中下，阻塞胃气通降道路，本方加赭石之质重下行，莱菔子之升降气机，开结通便，便通则胃气下行，呕吐自止。紫硇砂腐蚀瘤体，号称肿瘤克星，用量宜小，为防其使瘤体破裂出血，可加服儿茶。

## 医案4：脊髓神经胶质瘤

**来源**：《李可老中医急危重症疑难病经验专辑》
**患者**：女、19岁
**诊断**：C5—T3水平脊髓占位病变，神经胶质瘤（北京天坛医院MRI确诊）
**症状**：颈项强痛，脊柱向右侧弯，转侧困难，斜颈已6年。左肩背沉困重痛，四肢无力，左下肢肌萎缩，双下肢进行性麻木近半年已不知痛痒。左腿环跳穴及足跟部电击样阵痛，一日数发，步态蹒跚、倾侧，已休学2个月。面色白无华，气怯神倦，头目昏眩，瑟缩畏寒。
**辨证**：禀赋素虚，嗜食生冷，脾失健运，正气先虚，痰湿内生，瘀血内阻，寒伤督脉，真阳失运，日久湿痰死血，阻塞经脉，成为有形癥积。病在脊椎，属督脉为病，奇经八脉病变。
**处方**：
1. 生芪240克，葛根90克，麻黄15克（先煎去沫），附子30克，细辛20克，漂海藻、生甘草、生半夏、云苓各30克，白芍、川芎各30克，白芥子（炒研）、桃仁、红花、僵蚕、地龙、两头尖、子蜂房、天南星、高丽参（另炖）、灵脂各10克，鲜生姜30克，大枣12枚。加冷水1500毫升，文火煮取450毫升，3次分服，5剂。
2. 全虫尾15克，大蜈蚣20条，川贝、土元、炮甲珠各30克，麝香2克，共研细粉，分作15包，1包/次，3次/日，随中药服。
3. 夏枯草1500克，依法熬膏，10毫升/次，3次/日。
**剂量**：先后八诊，历时7个月，服汤剂107剂，扶正化瘤丸1料
**疗效**：药进5剂，每服皆得畅汗，颈项肩背沉困感遂去大半。服药47剂，诸证已去十之七八，下肢感觉渐复。MRI复查："C6—T4脊髓占位病变与原片比较，未见明显变化。"原病灶周围模糊阴影消失，边界清楚；原脊柱向右侧弯已恢复正常，斜颈已愈；患肢肌萎缩已恢复如初。6年来痛经痼疾亦愈，面色红润，精神饱满，考试成绩优秀。
**李可按语**：本病临床罕见，机理不明。从中医经典理论辨析，当属奇经八脉病变。缘由正气先虚，痰湿内生，寒伤督脉，真阳失运。治当养正消积，扶正温阳为先，遵伤寒、金匮之理，邪之来路，即邪之去路，以麻黄附子细辛汤深入少阴之里，透发伏寒，兼开太阳之表，开门逐盗，引邪外透。破格重用生芪之入督脉，补大气，益气运血，温通督脉。余治肿瘤40余年，深感中医经典理论生命力之强大。

## 医案5：宫颈鳞癌晚期

**来源**：《李可老中医急危重症疑难病经验专辑》
**患者**：女、52岁
**诊断**：宫颈鳞癌晚期，癌肿呈菜花样破溃，膀胱直肠浸润转移（晋中二院妇科活检确诊）
**症状**：出血日见增多，少腹憋胀，疼痛如绞，里急后重如痢，尿频尿急，带色青黄夹黑，秽臭，呕逆不思饮食，苔黄厚腻，脉弦滑劲急，双腹股沟淋巴结肿硬如枣，触痛。卧床不起。
**辨证**：肝郁气滞，湿热化毒结于胞宫
**处方**：
- 内服方：败酱草、白头翁、蚤休、蛇舌草、半枝莲、墓头回、川草薢、当归、刘寄奴、乌贼骨、酒大黄、土茯苓各30克，茜草炭18克，桂枝、桃仁、丹皮、赤芍、黄柏、甘草各15克。二煎混匀，取浓汁600毫升，日分3服。
- 外洗方：松、柏、槐、桑、嫩核桃树枝各30克，生军、甘草各30克，莪术60克，煎汤一大盆，熏洗坐浴。
**剂量**：两方各10剂，后续根据证情加减，前后经治13个月，服药210剂，外洗坐浴约90剂
**疗效**：10剂后宫颈癌中毒症状迅速消除，呕逆除，食纳增，出血渐少，腹痛、里急后重、尿频尿急均愈。经治4个月又25天，宫颈部菜花状破溃瘤体已大部脱落，仅剩小姆指甲盖大一块，腹股沟肿大之淋巴结消失。体重由37公斤增至63.5公斤，面上老年斑退净，部分白发变黑，皱纹消失。临床治愈，追访22年，活至80岁后无疾而终。
**李可按语**：宫颈癌多由生育或流产过多，房室不节，八脉损伤，累及肝脾肾，元气先虚为基本原因。"气郁"既是本病形成的重要因素，又可左右本病的进程。解郁之法，单靠药物是不行的，"心病还须心药医"。晚期宫颈癌治法：补法贯彻始终，以保护脾胃为第一要义；探索病机选方遣药，基础方选逍遥散合桂枝茯苓丸化裁。"炉烟虽熄，灰中有火"，即使临床妇检证实瘤体脱落，仍须丸方久服，养正消积。

## 医案6：小儿急性粒细胞型白血病

**来源**：《李可老中医急危重症疑难病经验专辑》
**患者**：男、13岁
**诊断**：急性粒细胞型白血病（山医三院住院号77—0723，血色素4克，白细胞36万）
**症状**：两月前突然高热寒战，体温40℃，鼻衄如注2日不止，大便如柏油状。面色萎黄虚浮，唇指白如麻纸，眩晕不能坐立，纳呆日仅进食1~2两，五心烦热，心动震衣，自汗如洗，两目失神，舌如去膜猪腰子，光绛无苔而干，六脉浮弦搏指，一息七至以上。
**辨证**：气阴两竭之死候
**处方**：生芪30克，当归、红参（另炖）、麦冬（小米拌炒）、五味子、三仙炭、炙草各10克，山萸肉、九地各30克，砂仁10克，元肉、女贞子、旱莲草各15克，阿胶18克（化入），鲜生姜5片，大枣6枚
**剂量**：浓煎，小量多次分服。首剂得效，连服10剂后改为补脾肾方续服，前后4个月，服药40剂
**疗效**：服3剂可起坐，服5剂后可进食半斤，头晕大减，精神转佳。服至第7剂已能下床散步，舌上布薄白苔，津润，胃气来复。日可进食1斤多。后出现泛呕泄泻、腰困如折等厥脱先兆，急以温脾固肾之品抢救，服2剂胃寒退，泻止脉敛，服5剂脐动隐，元阳固。血色素上升至9.5克，白细胞降至5万7千。继续调理，血色素11克，白细胞2万7千，稳步向愈。
**李可按语**：小儿白血病类似"小儿急痨"，初期邪毒炽盛，可借鉴温病治法，以清瘟败毒饮重用生石膏250~500克，一昼夜连服3大剂，即可阻断病势。若禀赋素虚，邪从寒化、虚化，当急急固脱为先。一切攻癌解毒、苦寒败胃之品，毫末不可沾唇。扶得一分正气，便退却一分邪气，保得一分胃气，便有一线生机。经抢救4个月，服药40剂，未用一味抗癌药，终于降服白细胞，使血色素恢复正常。可见"以人为本"的思想，固护脾肾元气的治则，在癌症治疗中具有特殊地位。

## 医案7：食道中段癌

**来源**：《李可老中医急危重症疑难病经验专辑》
**患者**：女、60岁（李可老母）
**诊断**：食道中段癌，放疗后并发梗阻
**症状**：水米不入已5天，以输液维持生命。血色素6克，白细胞3400，体重37.5公斤，一身大肉尽脱。气息奄奄，舌光剥，唇焦裂，眼眶塌陷。胸背刺痛不休，干渴，喉间如火焚，午后潮热。
**辨证**：湿痰死血，滞留经络；放疗伤阴
**处方**：
- 加味开道散：火硝30克，紫硇砂15克，明雄黄3克，硼砂15克，真落水沉香5克，枯矾6克，柿霜粉30克，煅礞石5克，冰片1.5克，乌梅肉15克，共研极细粉
- 主方：赭石粉50克，旋覆花（包）15克，白参（另炖）10克，生芪、当归、花粉、元参、沙参、生半夏各30克，炙草10克，姜汁10毫升，蜂蜜120克，蛇舌草120克，黄药子30克（后两味煎汤代水煎药）
**剂量**：开道散每次1克，蜜汁调糊缓缓含化，半小时许1次，日10余次；汤剂日分多次缓缓呷服。连服汤剂3年，达1千余剂。散剂终生未断。
**疗效**：梅花针叩刺胸背疼痛部位3日后，疼痛大为缓解，停用杜冷丁可入睡。含化散剂5天后可饮少许蜜水下咽，第15天试服牛奶1小杯顺利服下，攻克梗阻关。此后病情逐日缓解，返家60日后可顺利进食油茶、泡蒸馍、细挂面。带癌生存10多年。
**李可按语**：能否闯过疼痛、梗阻、出血、厥脱等四关，则是晚期食道癌病人生死存亡的关键。散刺出血法攻克疼痛关；开道散攻克梗阻关；来复汤合升陷汤攻克出血厥脱关。治癌是持久战，须长服培元固本散治本，拔除病根。

## 医案8：骨瘤（溶骨肉瘤）

**来源**：《李可老中医急危重症疑难病经验专辑》

# 李可攻癌2号方（晚年治肿瘤新突破）

> 来源：孙其新《李可医案处方集》第8章，基于45例医案300首处方（2007-2009年）整理

## 一、李可治肿瘤核心思想

### 1.1 以阴阳为纲，寒热虚实分型

李可后期治肿瘤，以阴阳为纲，将肿瘤分为两大类型：
- **阳虚寒凝型**：用攻癌2号方
- **阴证化阳型**：用攻癌基础方加鳖甲、蚤休、蒲公英（简称"攻癌基础鳖蚤英"）

### 1.2 组方思路：中医证 + 西医病 + 找原点

李可对中西医病证关系的经典论述：
> "中医之证是整体失调在患病阶段主要矛盾的集中点，西医之病是整体失调所派生的局部病变；治证就是调节整体，整体康复则局部病变常可奇迹般地不治而愈；当中医之证与西医之病发生冲突时，要毫不犹豫地舍病从证。"

"找原点"原则：哪个部位阳气不能到达，那个地方就有病，就治那个东西。哪个部位长瘤子，就用那个系统方子。

### 1.3 处方公式

```
（基本方 + 主治方） + 攻坚化瘤方
=（温阳散寒方 + 该系统方） + 攻癌基础方
```

- **基本方**（治本）：四逆汤、附桂理中汤、真武汤、麻黄附子细辛汤
- **主治方**（治本）：各系统方（见下表）
- **攻坚化瘤方**（治标）：攻癌基础方、攻癌夺命汤、阳和汤

### 1.4 各系统主治方速查

| 病位 | 主治方 |
|------|--------|
| 脑瘤 | 芎芷麻附细辛汤（麻黄10g、制附片、细辛各45g、川芎90g、白芷30g、炙甘草60g） |
| 肺癌 | 小青龙汤、阳和汤、千金苇茎汤、瓜蒌薤白丹参饮 |
| 食管癌 | 开道散、止呕汤 |
| 胃癌 | 参灵散、三畏汤、双呕汤 |
| 肠癌 | 赤油散（赤石脂、油桂）、薏苡附子败酱散、乌梅丸 |
| 肝癌 | 茵陈五苓散、茵陈术附汤、乌梅丸 |
| 胰腺癌 | 大柴胡汤、茵陈五苓散、薏苡附子败酱散 |
| 宫颈癌 | 桂枝茯苓丸、癃闭散、通淋散 |

## 六、李可治肿瘤分型总结

### 早期分型（《李可经验专辑》时期）
- **痰毒热化型**：攻癌夺命汤
- **痰凝寒化型**：攻癌基础方 + 麻附细辛汤

### 晚期分型（近10年）
- **阳虚寒凝型**：攻癌2号方
- **阴证化阳型**：攻癌基础方 + 鳖甲、蚤休、蒲公英

**关键转变**：李可晚期更重视温阳散寒，认为肿瘤的根本是阳虚。"肿瘤这个东西最早产生的阳虚，阳气虚了以后，慢慢就结成小块儿，然后逐渐长大，成为一个影响人们生命的东西。"

**张涵按语**：不知患者摔一跤与患肿瘤是巧合？还是体内瘀血误诊为肿瘤？此为张涵独立四诊在先、李可确认处方在后的典型案例。附有复旦大学附属肿瘤医院病理报告：右颈部非霍奇金B细胞淋巴瘤。
